# ArpCurses
###A scapy-based arp poisoning with curses interface.


#####How to launch from source
  1. Install python-scapy.
  2.   ```sudo python main.py```

#####How to freeze packages("compile")
  1. Install python-scapy.
  2. Install pyInstaller.
  3.  ```pyinstaller -F main.py ```
  4.  ```sudo ./dist/main```

#####Compatibility
  1. GNU/Linux
  2. OsX
